Layer 기반 시각 패턴 해석 1. 문제 제기 

기존의 카르노맵 해석은 각 셀의 논리값과 인접 그룹에 초점을 맞추는 경우가 많다. 그러나 본 연구에서 관찰된 점 패턴, 모서리 패턴, 고리 패턴, 체커보드 패턴은 단순한 셀의 집합이라기보다 특정한 구조를 가진 시각적 형태로 인식된다.

특히 Exactly-2 함수에서 나타나는 고리 패턴은 단순히 Weight=2 상태가 여섯 개 존재한다는 사실만으로는 충분히 설명되지 않는다. 왜 동일한 여섯 개의 상태가 카르노맵 위에서 고리처럼 보이는지에 대한 추가적인 설명이 필요하다.

본 절에서는 이러한 패턴을 Boolean 공간의 Layer 구조 관점에서 해석한다.

2. Boolean 공간의 Layer 구조 

4변수 Boolean 공간은 총 16개의 상태로 구성된다.

각 상태는 포함된 1의 개수(Hamming Weight)에 따라 다음 다섯 개의 층으로 분류할 수 있다.

Layer 0 : Weight = 0 Layer 1 : Weight = 1 Layer 2 : Weight = 2 Layer 3 : Weight = 3 Layer 4 : Weight = 4 

각 층에 포함되는 상태 수는 다음과 같다.

Layer 상태 수 L0 1 L1 4 L2 6 L3 4 L4 1 

이는 조합론적으로 다음 식으로 표현된다.

[ |L_k| = \binom{4}{k} ]

따라서 4변수 Boolean 공간은 1-4-6-4-1 구조를 가진다.

3. Layer와 시각 패턴의 대응 

완전대칭함수는 변수의 위치와 무관하게 Weight만으로 출력이 결정된다.

따라서 완전대칭함수는 특정 Layer를 선택하는 함수로 해석할 수 있다.

함수 선택 Layer Exactly-0 L0 Exactly-1 L1 Exactly-2 L2 Exactly-3 L3 Exactly-4 L4 XOR L1 ∪ L3 XNOR L0 ∪ L2 ∪ L4 

이러한 관점에서 카르노맵의 시각 패턴은 Layer 선택 결과의 시각적 표현으로 해석할 수 있다.

4. Exactly-2와 고리 패턴 

Exactly-2 함수는 Weight가 정확히 2인 상태만 선택한다.

Weight=2 상태는 총 6개 존재하며, 이는 4변수 Boolean 공간에서 가장 큰 Layer를 형성한다.

중요한 점은 고리 패턴의 원인이 Weight=2 상태의 개수 자체에 있는 것이 아니라는 사실이다.

고리 패턴은 다음 두 요소가 결합되어 형성된다.

Weight=2 Layer의 구조 Gray Code 기반 0132 배열 

Weight=2 상태들이 0132 배열 위에 배치되면 특정 영역을 따라 연결된 띠 형태를 형성하게 된다. 인간은 이러한 띠 구조를 시각적으로 하나의 폐곡선 또는 고리 구조로 인식할 수 있다.

따라서 고리 패턴은 Layer 구조와 배열 방식이 결합되어 나타나는 결과로 해석할 수 있다.

5. 시각 패턴은 Layer의 투영인가 

본 연구에서 관찰된 점 패턴, 모서리 패턴, 고리 패턴은 모두 특정 Layer가 카르노맵 평면에 투영된 결과로 해석할 수 있다.

즉,

Boolean 공간의 Layer 구조

↓

카르노맵 배열

↓

시각 패턴

이라는 대응 관계가 존재한다.

이 관점에서 카르노맵은 단순한 논리 최소화 도구가 아니라 Boolean 공간의 구조를 2차원 평면에 시각화하는 도구로 해석될 수 있다.

6. 연구 가설 

본 연구는 다음 가설을 제안한다.

카르노맵의 주요 시각 패턴은 Boolean 공간의 Layer 구조가 2차원 평면에 투영된 결과이다.

이 가설이 타당하다면, 카르노맵에 나타나는 다양한 패턴은 개별 함수의 우연한 배열이 아니라 Boolean 공간의 구조적 특성을 반영하는 현상으로 이해할 수 있다.
