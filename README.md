그리고 가장 중요한 지시사항은 이것 같아.
이 논문은 기존 「카르노맵의 시각적 패턴 기반 논리회로 학습 방법 연구」의 일부로 병합하지 말 것.
독립 논문으로 유지할 것.
본 논문의 핵심 기여는 XOR/XNOR 학습법이 아니라 4변수 완전대칭함수의 시각 패턴 분류와 Hamming Weight 층 구조를 이용한 해석에 있음.
특히 Exactly-0~4, XOR, XNOR의 패턴 분류 체계와 XOR 체커보드의 구조적 해석을 중심으로 문서를 재구성할 것.
