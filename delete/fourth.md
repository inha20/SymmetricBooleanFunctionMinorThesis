
초록 

카르노맵(Karnaugh Map)은 일반적으로 논리함수의 최소화를 위한 도구로 사용되며, 교육 현장에서도 SOP 및 POS 형태의 간소화 과정에 초점을 맞추어 설명되는 경우가 많다. 그러나 실제 카르노맵에는 체커보드 패턴, 이중대각 패턴, 고리 패턴 등 다양한 시각적 구조가 나타나며, 이러한 구조는 단순한 배열 결과가 아니라 Boolean 함수의 구조적 특성을 반영할 가능성이 있다.

본 연구에서는 4변수 카르노맵에 나타나는 완전대칭함수(Symmetric Boolean Function)의 시각 패턴을 분석하고, 이를 Hamming Weight 기반 Layer 구조의 관점에서 해석하였다. 특히 Exactly-k 함수와 XOR/XNOR 함수를 대상으로 Gray Code 배열(0132)과 일반 이진 배열(0123)을 비교하여 시각 패턴과 구조적 특성의 관계를 분석하였다.

분석 결과, Exactly-0 및 Exactly-4 함수는 점 패턴, Exactly-1 및 Exactly-3 함수는 모서리 패턴, Exactly-2 함수는 고리 패턴을 형성하였다. 또한 XOR 및 XNOR 함수는 배열 방식에 따라 체커보드 패턴 또는 이중대각 패턴으로 표현되었다. 특히 XOR 체커보드는 홀수 Layer 선택 구조와 Gray Code 배열의 인접성 보존 성질이 결합되어 형성되는 현상으로 해석할 수 있음을 확인하였다.

본 연구는 완전대칭함수를 특정 Layer 또는 Layer 집합을 선택하는 함수로 해석할 수 있음을 보였으며, 카르노맵의 시각 패턴이 Layer 구조와 배열 방식의 상호작용 결과로 이해될 수 있음을 제안한다. 이를 통해 카르노맵을 단순한 논리 최소화 도구가 아니라 Boolean 공간 구조를 시각적으로 표현하는 도구로 해석하는 새로운 관점을 제시한다.





2. 이론적 배경 2.1 카르노맵의 구조 

카르노맵(Karnaugh Map)은 Boolean 함수를 시각적으로 표현하여 논리식의 최소화를 수행하기 위한 도구이다.

4변수 카르노맵은 총 16개의 셀로 구성되며, 각 셀은 입력 변수 A, B, C, D의 하나의 상태에 대응한다.

일반적인 진리표는 이진수 순서에 따라 상태를 배열하지만, 카르노맵은 인접한 상태가 하나의 변수만 다르도록 배치된다.

이를 통해 인접 셀을 그룹화하여 논리식을 간단하게 표현할 수 있다.

기존 연구와 교육에서는 이러한 최소화 기능에 주목하는 경우가 많다.

그러나 카르노맵은 단순한 계산 도구를 넘어 Boolean 공간의 구조를 시각적으로 표현하는 도구로도 해석할 수 있다.

본 연구에서는 카르노맵을 Boolean 공간의 2차원 표현으로 간주하고, 그 위에 나타나는 시각 패턴을 분석 대상으로 삼는다.

2.2 Gray Code 배열과 일반 이진 배열 

4변수 카르노맵은 일반적으로 Gray Code 순서를 사용한다.

행과 열은 다음과 같이 배치된다.

00, 01, 11, 10

이 배열은 인접한 두 상태가 정확히 하나의 비트만 다르도록 설계되어 있다.

즉 인접 상태 사이의 Hamming Distance가 항상 1이 된다.

본 연구에서는 이를 Gray Code 배열(0132 배열)이라 부른다.

반면 일반적인 이진 배열은 다음과 같다.

00, 01, 10, 11

본 연구에서는 이를 0123 배열이라 부른다.

동일한 Boolean 함수라도 배열 방식에 따라 서로 다른 시각 패턴이 나타날 수 있다.

예를 들어 XOR 함수는

0132 배열에서는 체커보드 패턴 0123 배열에서는 이중대각 패턴 

으로 표현된다.

따라서 카르노맵의 시각 패턴을 이해하기 위해서는 함수뿐 아니라 배열 방식도 함께 고려할 필요가 있다.

2.3 완전대칭함수와 Hamming Weight 

Boolean 함수가 변수의 순서와 무관하게 동일한 출력을 유지하는 경우 이를 완전대칭함수(Symmetric Boolean Function)라 한다.

예를 들어

A⊕B⊕C⊕D

는 변수의 순서를 어떻게 바꾸어도 동일한 결과를 생성한다.

완전대칭함수는 개별 변수의 위치보다 입력에 포함된 1의 개수에 의해 결정되는 경우가 많다.

본 연구에서는 입력 상태의 Hamming Weight를 다음과 같이 정의한다.

[ w=A+B+C+D ]

여기서 w는 입력 상태에 포함된 1의 개수를 의미한다.

4변수 Boolean 공간에서는 가능한 Weight가

0, 1, 2, 3, 4

의 다섯 값으로 구분된다.

각 Weight에 해당하는 상태 수는 다음과 같다.

[ 1,4,6,4,1 ]

이는 조합론적으로

[ \binom{4}{k} ]

와 일치한다.

본 연구에서는 이러한 Weight 구조를 이용하여 카르노맵 패턴을 해석한다.

2.4 Boolean 공간의 Layer 구조 

4변수 Boolean 공간은 Hamming Weight를 기준으로 다섯 개 Layer로 분류할 수 있다.

Layer Weight 상태 수 L₀ 0 1 L₁ 1 4 L₂ 2 6 L₃ 3 4 L₄ 4 1 

이 구조는

[ 1-4-6-4-1 ]

형태를 가지며 파스칼 삼각형의 계수와 대응된다.

본 연구에서는 Layer 구조를 Boolean 공간의 기본 골격으로 해석한다.

완전대칭함수는 개별 상태를 선택하는 함수가 아니라 특정 Layer 또는 Layer 집합을 선택하는 함수로 이해할 수 있다.

예를 들어

Exactly-2 함수는

[ L_2 ]

를 선택한다.

XOR 함수는

[ L_1 \cup L_3 ]

를 선택한다.

XNOR 함수는

[ L_0 \cup L_2 \cup L_4 ]

를 선택한다.

따라서 완전대칭함수의 구조는 Layer 선택 문제로 재해석할 수 있다.

2.5 Layer 구조와 시각 패턴 

본 연구는 카르노맵의 시각 패턴이 Layer 구조와 관련되어 있다는 가설을 제안한다.

동일한 함수는 항상 동일한 Layer 구조를 가진다.

그러나 카르노맵에서 관찰되는 구체적인 시각 패턴은 배열 방식에 따라 달라질 수 있다.

예를 들어 XOR 함수는 항상

[ L_1 \cup L_3 ]

를 선택하지만,

0132 배열에서는 체커보드 패턴 0123 배열에서는 이중대각 패턴 

으로 나타난다.

이 사실은 Layer 구조와 시각 패턴을 구분해야 함을 의미한다.

본 연구에서는

Layer 구조

↓

배열 방식

↓

시각 패턴

이라는 해석 틀을 사용한다.

즉 Layer 구조는 함수의 본질적 특성을 나타내며, 시각 패턴은 그 구조가 특정 배열 방식 위에 표현된 결과로 해석한다.







3. 시각 패턴의 정의 3.1 시각적 대칭 

본 연구에서는 카르노맵에 나타나는 구조를 분석하기 위하여 시각적 대칭(Visual Symmetry)의 개념을 정의한다.

시각적 대칭이란 특정 축에 대하여 카르노맵을 반사하였을 때 대응되는 위치의 값이 동일하게 유지되는 성질을 의미한다.

예를 들어 세로축 대칭은

F(a,b)=F(a,5-b)

가로축 대칭은

F(a,b)=F(5-a,b)

주대각선 대칭은

F(a,b)=F(b,a)

로 표현할 수 있다.

본 연구에서는 이러한 관계가 성립할 때 해당 축에 대한 시각적 대칭이 존재한다고 정의한다.

3.2 시각적 동일 

시각적 동일(Visual Repetition)은 대칭과 구별되는 개념이다.

시각적 동일이란 특정 부분 패턴이 반복적으로 나타나는 현상을 의미한다.

예를 들어 XOR 체커보드 패턴에서는 동일한 구조가 평면 전체에 반복적으로 나타난다.

시각적 대칭이 반사 관계를 의미한다면, 시각적 동일은 반복 관계를 의미한다.

따라서 두 개념은 독립적으로 존재할 수 있다.

3.3 체커보드 패턴 

체커보드 패턴(Checkerboard Pattern)은 인접한 셀의 값이 항상 서로 반대가 되는 패턴을 의미한다.

예를 들면 다음과 같다.

0101

1010

0101

1010

4변수 XOR 함수를 Gray Code 배열(0132)로 표현하면 이러한 구조가 나타난다.

Gray Code 배열에서는 인접 셀이 정확히 하나의 변수만 다르다.

따라서 인접 이동 시 Hamming Weight의 홀짝성이 반드시 변하게 된다.

XOR 함수는 홀수 Weight 층만 선택하므로 인접 셀의 출력값이 항상 반전된다.

그 결과 체커보드 패턴이 형성된다.

본 연구에서는 이를 인접성 기반 패턴(adjacency-based pattern)으로 분류한다.

3.4 이중대각 패턴 

일반 이진 배열(0123)에서 XOR 함수를 표현하면 다음과 같은 패턴이 나타난다.

0110

1001

1001

0110

이 패턴은 주대각선과 부대각선에 대한 대칭성을 동시에 가진다.

본 연구에서는 이를 이중대각 패턴(Double-Diagonal Pattern)이라 정의한다.

체커보드 패턴이 인접성 구조를 강조한다면, 이중대각 패턴은 XOR 함수의 대칭 구조를 강조하는 표현이라 볼 수 있다.

3.5 고리 패턴 

Exactly-2 함수는 Weight가 정확히 2인 상태만 선택하는 함수이다.

4변수 Boolean 공간에서 Weight=2 상태는 총 6개 존재한다.

이를 Gray Code 기반 0132 배열에 배치하면 선택된 상태들이 띠(band) 형태의 연결 구조를 형성한다.

인간은 이러한 연결 구조를 하나의 폐곡선 또는 고리와 유사한 형태로 인식할 수 있다.

본 연구에서는 이를 고리 패턴(Ring Pattern)이라 정의한다.

고리 패턴은 단순히 Weight=2 상태가 가장 많은 층이라는 사실만으로 설명되지 않는다.

오히려 Weight=2 Layer의 구조와 Gray Code 배열의 인접성 보존 성질이 결합되어 나타나는 시각적 현상으로 해석할 수 있다.

3.6 시각 패턴 분류 체계 

본 연구에서 관찰된 주요 시각 패턴은 다음과 같이 분류할 수 있다.

패턴 대표 함수 점 패턴 Exactly-0, Exactly-4 모서리 패턴 Exactly-1, Exactly-3 고리 패턴 Exactly-2 체커보드 패턴 XOR, XNOR (0132 배열) 이중대각 패턴 XOR, XNOR (0123 배열) 

이 분류는 논리식이 아니라 카르노맵 위에 나타나는 시각적 구조를 기준으로 수행한 것이다.

이후 장에서는 각 패턴이 Boolean 공간의 Layer 구조와 어떤 관계를 가지는지 분석한다.

4. 완전대칭함수의 시각 패턴 분석 4.1 분석 대상 

본 연구에서는 입력 변수의 순서에 영향을 받지 않는 완전대칭함수(Symmetric Boolean Function)를 대상으로 시각 패턴을 분석하였다.

특히 Hamming Weight만으로 출력이 결정되는 다음 함수들을 분석 대상으로 선정하였다.

Exactly-0 Exactly-1 Exactly-2 Exactly-3 Exactly-4 XOR XNOR 

이 함수들은 모두 Weight 층 구조와 직접적으로 연결되며, Layer 기반 해석이 가능하다.

4.2 Exactly-0 함수 

Exactly-0 함수는 모든 입력이 0인 경우에만 출력이 1이 된다.

즉,

[ L_0 ]

만을 선택한다.

선택되는 상태는 단 하나뿐이므로 카르노맵에서는 하나의 점으로 표현된다.

본 연구에서는 이를 점 패턴(Point Pattern)으로 분류한다.

4.3 Exactly-1 함수 

Exactly-1 함수는 입력 중 정확히 하나만 1인 경우 출력이 1이 된다.

즉,

[ L_1 ]

을 선택한다.

Weight=1 상태는 총 4개 존재하며 카르노맵에서는 서로 떨어진 위치에 배치된다.

그 결과 네 개의 점이 모서리 방향으로 분산된 구조가 나타난다.

본 연구에서는 이를 모서리 패턴(Corner Pattern)으로 분류한다.

4.4 Exactly-2 함수 

Exactly-2 함수는 Weight가 정확히 2인 상태만 선택한다.

즉,

[ L_2 ]

를 선택한다.

Weight=2 상태는 총 6개 존재하며 4변수 Boolean 공간에서 가장 큰 Layer를 형성한다.

그러나 고리 패턴의 원인이 단순히 상태 수가 많기 때문이라고 보기는 어렵다.

Weight=2 상태들을 Gray Code 기반 0132 배열에 배치하면 선택된 셀들이 연결된 띠 형태를 형성한다.

인간은 이 구조를 하나의 폐곡선 또는 고리와 유사한 형태로 인식할 수 있다.

따라서 Exactly-2의 고리 패턴은

Weight=2 Layer의 구조 Gray Code 배열의 인접성 보존 성질 

이 결합된 결과로 해석할 수 있다.

4.5 Exactly-3 함수 

Exactly-3 함수는 Weight가 정확히 3인 상태를 선택한다.

즉,

[ L_3 ]

을 선택한다.

Weight=3 상태 수는 Weight=1 상태 수와 동일하다.

또한 카르노맵에서는 Exactly-1과 동일한 시각 구조를 형성한다.

따라서 Exactly-1과 Exactly-3은 서로 다른 Layer를 선택하지만 동일한 모서리 패턴으로 분류할 수 있다.

4.6 Exactly-4 함수 

Exactly-4 함수는 모든 입력이 1인 경우에만 출력이 1이 된다.

즉,

[ L_4 ]

만을 선택한다.

선택되는 상태 수는 1개이며 Exactly-0과 동일한 점 패턴을 형성한다.

따라서 Exactly-0과 Exactly-4는 서로 다른 Layer에 대응하지만 동일한 시각 패턴으로 분류된다.

4.7 XOR 함수 

XOR 함수는 Weight가 홀수인 상태에서 출력이 1이 된다.

즉,

[ L_1 \cup L_3 ]

을 선택한다.

0132 배열에서는 체커보드 패턴이 나타난다.

반면 0123 배열에서는 이중대각 패턴이 나타난다.

이는 동일한 Layer 구조가 배열 방식에 따라 서로 다른 시각 패턴으로 표현될 수 있음을 보여준다.

그러나 두 경우 모두 동일한 홀수 Layer 선택 구조를 기반으로 한다는 점에서, 패턴의 근본 원인은 Layer 구조에 있다고 해석할 수 있다.

4.8 XNOR 함수 

XNOR 함수는 Weight가 짝수인 상태에서 출력이 1이 된다.

즉,

[ L_0 \cup L_2 \cup L_4 ]

를 선택한다.

이는 XOR 함수의 보수 구조에 해당한다.

0132 배열에서는 XOR의 반전된 체커보드 패턴이 나타난다.

0123 배열에서는 XOR과 대응되는 이중대각 패턴이 나타난다.

따라서 XNOR 역시 Layer 선택 구조와 배열 방식의 상호작용으로 시각 패턴이 형성된다고 해석할 수 있다.

4.9 분석 결과 요약 

분석 결과 완전대칭함수의 시각 패턴은 다음과 같이 정리할 수 있다.

함수 선택 Layer 시각 패턴 Exactly-0 L₀ 점 패턴 Exactly-1 L₁ 모서리 패턴 Exactly-2 L₂ 고리 패턴 Exactly-3 L₃ 모서리 패턴 Exactly-4 L₄ 점 패턴 XOR L₁ ∪ L₃ 체커보드/이중대각 XNOR L₀ ∪ L₂ ∪ L₄ 체커보드/이중대각 

이 결과는 카르노맵의 시각 패턴이 단순한 배열 결과가 아니라 Layer 구조와 표현 방식의 상호작용을 반영한다는 사실을 시사한다.

5. XOR/XNOR 패턴의 구조적 해석 5.1 XOR 함수와 Layer 구조 

4변수 XOR 함수는 다음과 같이 정의된다.

[ A \oplus B \oplus C \oplus D ]

XOR 함수는 입력 상태에 포함된 1의 개수가 홀수일 때 출력이 1이 된다.

즉 XOR 함수는 다음 Layer를 선택한다.

[ L_1 \cup L_3 ]

반면

[ L_0,;L_2,;L_4 ]

는 선택되지 않는다.

따라서 XOR 함수는 Boolean 공간의 홀수 Weight 층을 선택하는 함수로 해석할 수 있다.

5.2 Gray Code 배열의 인접성 보존 

4변수 카르노맵은 일반적으로 Gray Code 배열을 사용한다.

행과 열은

00, 01, 11, 10

순서로 배열된다.

Gray Code의 핵심 성질은 인접한 두 상태가 정확히 하나의 비트만 다르다는 점이다.

즉 인접 상태 사이의 Hamming Distance는 항상 1이다.

따라서 카르노맵에서 상하좌우로 한 칸 이동하는 것은 Boolean 공간에서 하나의 변수만 변경하는 것과 대응된다.

본 연구에서는 이를 Gray Code 배열의 인접성 보존 성질이라 부른다.

5.3 인접 이동과 Weight 변화 

Boolean 공간에서 하나의 비트가 변경되면 Hamming Weight는 반드시 1만큼 증가하거나 감소한다.

즉

[ L_0 \leftrightarrow L_1 ]

[ L_1 \leftrightarrow L_2 ]

[ L_2 \leftrightarrow L_3 ]

[ L_3 \leftrightarrow L_4 ]

사이에서만 이동할 수 있다.

따라서 인접한 두 상태는 항상 Weight의 홀짝성이 반대이다.

이 성질은 XOR 체커보드 패턴 형성의 핵심 원인이 된다.

5.4 XOR 체커보드 패턴의 형성 

XOR 함수는 홀수 Layer만 선택한다.

[ L_1 \cup L_3 ]

Gray Code 배열에서는 인접 상태가 항상 한 비트만 다르다.

따라서 인접 이동 시 Weight의 홀짝성이 반드시 반전된다.

결과적으로 인접한 두 셀은 항상 서로 다른 출력값을 가진다.

즉

[ 0 \leftrightarrow 1 ]

관계가 모든 인접 셀에서 성립한다.

그 결과 XOR 함수는 체커보드 패턴을 형성하게 된다.

따라서 체커보드 패턴은 단순한 시각적 우연이 아니라 다음 구조의 결과로 해석할 수 있다.

XOR

↓

홀수 Layer 선택

↓

Gray Code 인접성

↓

인접 셀 출력 반전

↓

체커보드 패턴

5.5 XNOR 함수의 해석 

XNOR 함수는 XOR 함수의 보수이다.

따라서

[ L_0 \cup L_2 \cup L_4 ]

를 선택한다.

즉 짝수 Weight 층만 선택하는 함수로 해석할 수 있다.

Gray Code 배열에서는 인접 상태의 Weight 홀짝성이 항상 반대가 되므로 인접 셀의 출력 역시 항상 반전된다.

따라서 XNOR 함수 역시 체커보드 패턴을 형성한다.

XNOR 체커보드는 XOR 체커보드의 반전 형태로 이해할 수 있다.

5.6 체커보드 패턴은 왜 특별한가 

본 연구에서 분석한 완전대칭함수들은 모두 Layer 구조로 표현할 수 있었다.

그러나 모든 함수가 체커보드 패턴을 형성하지는 않았다.

함수 선택 Layer 대표 패턴 Exactly-0 L₀ 점 Exactly-1 L₁ 모서리 Exactly-2 L₂ 고리 Exactly-3 L₃ 모서리 Exactly-4 L₄ 점 XOR L₁∪L₃ 체커보드 XNOR L₀∪L₂∪L₄ 체커보드 

체커보드 패턴은 완전대칭함수의 일반적 특징이 아니다.

체커보드는 홀수 Layer 전체 또는 짝수 Layer 전체를 선택하는 패리티(parity) 구조에서 나타나는 특수한 현상이다.

5.7 Layer 관점에서의 재해석 

기존 설명에서는 XOR 체커보드를 XOR 함수의 특수한 시각적 특징으로 이해하는 경우가 많았다.

그러나 본 연구는 이를 보다 일반적인 Layer 구조의 관점에서 해석한다.

체커보드는 XOR 자체에서 직접 발생하는 것이 아니라

Layer 선택 구조 Gray Code 배열의 인접성 보존 성질 

이 결합되어 나타나는 결과이다.

따라서 체커보드는 Boolean 공간의 구조가 2차원 카르노맵 위에 표현된 사례로 볼 수 있다.

5.8 소결론 

XOR 및 XNOR 함수의 체커보드 패턴은 단순한 시각적 특징이 아니다.

이는 Boolean 공간의 Layer 구조와 Gray Code 배열의 인접성 보존 성질이 결합되어 나타나는 구조적 현상이다.

본 연구의 결과는 카르노맵이 단순한 논리 최소화 도구를 넘어 Boolean 공간 구조를 시각적으로 표현하는 도구로 해석될 수 있음을 시사한다.

7. 결론 

본 연구에서는 4변수 카르노맵에 나타나는 완전대칭함수의 시각 패턴을 분석하고, 이러한 패턴이 Boolean 공간의 구조와 어떠한 관계를 가지는지 탐구하였다.

기존의 카르노맵 교육은 주로 논리식 최소화 과정에 초점을 두고 있으며, 카르노맵에 나타나는 시각적 구조는 상대적으로 적게 다루어져 왔다. 그러나 실제 카르노맵에는 점 패턴, 모서리 패턴, 고리 패턴, 체커보드 패턴, 이중대각 패턴 등 다양한 시각 구조가 존재하며, 이러한 구조는 단순한 시각적 우연이 아니라 함수의 논리적 특성과 관련될 가능성이 있다.

본 연구에서는 완전대칭함수를 Hamming Weight 기반 Layer 구조의 관점에서 해석하였다.

4변수 Boolean 공간은 Weight에 따라 L₀, L₁, L₂, L₃, L₄의 다섯 Layer로 구분할 수 있으며, 완전대칭함수는 특정 Layer 또는 Layer 집합을 선택하는 함수로 해석할 수 있음을 보였다.

분석 결과 다음과 같은 대응 관계를 확인하였다.

Exactly-0, Exactly-4 → 점 패턴 Exactly-1, Exactly-3 → 모서리 패턴 Exactly-2 → 고리 패턴 XOR, XNOR → 체커보드 또는 이중대각 패턴 

특히 XOR 함수는 L₁∪L₃를 선택하는 구조로 해석할 수 있으며, XNOR 함수는 L₀∪L₂∪L₄를 선택하는 구조로 해석할 수 있음을 확인하였다.

또한 XOR 체커보드 패턴은 단순히 XOR 함수의 특수한 시각적 형태가 아니라,

홀수 Layer 선택

↓

Gray Code 배열의 인접성 보존

↓

인접 셀 출력 반전

↓

체커보드 패턴

이라는 구조적 과정을 통해 형성됨을 보였다.

이는 체커보드 패턴이 XOR 함수의 우연한 배열 결과가 아니라 Boolean 공간 구조가 시각적으로 표현된 결과임을 시사한다.

본 연구는 또한 동일한 Layer 구조가 배열 방식에 따라 서로 다른 시각 패턴으로 나타날 수 있음을 확인하였다.

예를 들어 XOR 함수는 동일한 Layer 구조를 가지지만,

0132 배열에서는 체커보드 패턴 0123 배열에서는 이중대각 패턴 

으로 표현된다.

따라서 시각 패턴은 함수 자체에 의해서만 결정되는 것이 아니라 Layer 구조와 배열 방식의 상호작용에 의해 형성된다고 해석할 수 있다.

본 연구는 이러한 결과를 바탕으로 다음과 같은 탐색적 가설을 제안한다.

카르노맵의 주요 시각 패턴은 Boolean 공간의 Layer 구조가 특정 배열 방식 위에 투영된 결과일 수 있다.

만약 이 가설이 타당하다면, 점 패턴, 모서리 패턴, 고리 패턴, 체커보드 패턴 등은 서로 독립적인 현상이 아니라 하나의 공통 원리로 설명될 수 있다.

즉 카르노맵은 단순한 논리 최소화 도구를 넘어 Boolean 공간의 구조를 시각적으로 표현하는 도구로 해석될 수 있다.

향후 연구에서는 변수 재배열에 따른 구조 불변성, 카르노맵의 대칭 구조, 5변수 이상의 Boolean 공간에 대한 Layer 분석, 그리고 인간의 시각적 구조 인식 과정과의 관계 등을 탐구할 필요가 있다.

특히 Layer 구조와 시각 패턴의 대응 관계가 보다 일반적인 Boolean 함수 집합에서도 유지되는지에 대한 검증이 향후 중요한 연구 과제가 될 것으로 기대된다.

6. 배열 방식에 따른 시각 패턴 변화 6.1 동일한 함수와 서로 다른 패턴 

카르노맵의 시각 패턴은 함수 자체에 의해서만 결정되지 않는다.

동일한 Boolean 함수라도 배열 방식에 따라 서로 다른 시각 구조로 표현될 수 있다.

이를 확인하기 위하여 본 연구에서는 XOR 함수와 Exactly-2 함수를 Gray Code 배열(0132)과 일반 이진 배열(0123)로 각각 표현하여 비교하였다.

분석 결과 동일한 함수가 서로 다른 시각 패턴을 형성함을 확인하였다.

이는 카르노맵의 시각 패턴이 함수와 배열 방식의 상호작용에 의해 결정됨을 시사한다.

6.2 XOR 함수의 경우 

XOR 함수는 홀수 Weight 층을 선택하는 함수이다.

[ L_1 \cup L_3 ]

이 Layer 구조는 배열 방식과 무관하게 항상 동일하다.

그러나 시각적 표현은 배열 방식에 따라 달라진다.

0132 배열 

Gray Code 배열에서는 인접한 두 셀이 항상 하나의 변수만 다르다.

따라서 인접 이동 시 Weight의 홀짝성이 반드시 반전된다.

그 결과 XOR 함수는 체커보드 패턴을 형성한다.

0123 배열 

일반 이진 배열에서는 인접성이 Boolean 공간의 인접성과 일치하지 않는다.

따라서 체커보드 구조는 사라지고, 대신 대각선 방향의 대칭성이 두드러지게 나타난다.

본 연구에서는 이를 이중대각 패턴(Double-Diagonal Pattern)이라 정의하였다.

즉 동일한 XOR 함수가 배열 방식에 따라 서로 다른 시각 구조를 형성한다.

6.3 Exactly-2 함수의 경우 

Exactly-2 함수는

[ L_2 ]

를 선택한다.

이는 배열 방식과 무관하게 동일하다.

그러나 Weight=2 상태들이 평면 위에 배치되는 방식은 배열에 따라 달라진다.

0132 배열 

Gray Code 배열에서는 Weight=2 상태들이 연결된 띠 형태를 형성한다.

인간은 이를 하나의 폐곡선 또는 고리와 유사한 구조로 인식할 수 있다.

따라서 고리 패턴이 나타난다.

0123 배열 

일반 이진 배열에서는 동일한 Weight=2 상태들이 다른 위치에 배치된다.

이 경우 0132 배열에서 보이던 고리 형태는 약해지며, 패턴은 보다 분산된 형태로 나타난다.

즉 고리 패턴은 Weight=2 Layer 자체의 성질이라기보다 Layer 구조가 특정 배열 방식 위에 표현된 결과로 해석할 수 있다.

6.4 패턴은 함수의 성질인가 표현의 성질인가 

본 연구 과정에서 다음과 같은 질문이 제기되었다.

패턴은 함수의 성질인가, 표현의 성질인가?

분석 결과 두 요소 모두가 중요함을 확인하였다.

함수는 어떤 Layer를 선택할 것인지를 결정한다.

배열 방식은 그 Layer 구조를 어떻게 시각적으로 표현할 것인지를 결정한다.

따라서 함수만으로는 시각 패턴을 설명할 수 없으며, 배열 방식만으로도 시각 패턴을 설명할 수 없다.

시각 패턴은 두 요소의 결합 결과로 이해할 수 있다.

6.5 Layer 구조와 배열 방식

함수

↓

여기서 Layer 구조는 함수의 본질적 특성을 나타낸다.

반면 시각 패턴은 인간이 실제로 관찰하는 표현 결과이다.

예를 들어 XOR 함수의 경우

[ L_1 \cup L_3 ]

이라는 Layer 구조는 변하지 않지만,

0132 배열에서는 체커보드 0123 배열에서는 이중대각 

으로 표현된다.

따라서 Layer 구조와 시각 패턴은 구별되어야 한다.

6.6 Gray Code 배열의 역할 

Gray Code 배열은 카르노맵 최소화를 위해 설계된 배열 방식으로 알려져 있다.

그러나 본 연구 결과는 Gray Code 배열이 단순히 최소화만을 위한 배열이 아닐 가능성을 시사한다.

Gray Code 배열은 Boolean 공간의 인접성 구조를 보존한다.

그 결과 Layer 구조에 포함된 상태들의 관계가 시각적으로 드러나기 쉬워진다.

예를 들어 XOR의 체커보드 패턴과 Exactly-2의 고리 패턴은 모두 Gray Code 배열에서 보다 명확하게 나타난다.

따라서 Gray Code 배열은 Boolean 공간 구조를 시각적으로 표현하는 역할도 수행할 가능성이 있다.

6.7 소결론 

동일한 Boolean 함수라도 배열 방식에 따라 서로 다른 시각 패턴이 나타날 수 있다.
