배열 방식에 따른 Layer 패턴 변화 - 0132 배열과 0123 배열의 비교 - 1. 문제 제기 

본 연구에서는 Exactly-k 함수와 XOR/XNOR 함수를 분석하는 과정에서 동일한 Boolean 함수가 배열 방식에 따라 서로 다른 시각 패턴을 형성한다는 사실을 확인하였다.

대표적인 사례가 Exactly-2 함수와 XOR 함수이다.

Exactly-2 함수는 Gray Code 기반 0132 배열에서는 고리(Ring) 형태로 인식되지만, 일반 이진 배열인 0123 배열에서는 다른 형태로 나타난다.

또한 XOR 함수는 0132 배열에서는 체커보드 패턴으로 나타나지만, 0123 배열에서는 이중대각(Double-Diagonal) 패턴으로 나타난다.

이는 다음과 같은 질문을 제기한다.

카르노맵의 시각 패턴은 함수의 성질인가, 배열 방식의 성질인가?

본 절에서는 이 문제를 Layer 구조와 배열 방식의 상호작용 관점에서 분석한다.

2. Layer 구조와 배열 방식의 구분 

Boolean 함수가 선택하는 상태 집합은 함수 자체에 의해 결정된다.

예를 들어 Exactly-2 함수는 항상 Weight=2 상태만 선택한다.

[ L_2 ]

XOR 함수는 항상 홀수 Weight 층을 선택한다.

[ L_1 \cup L_3 ]

XNOR 함수는 항상 짝수 Weight 층을 선택한다.

[ L_0 \cup L_2 \cup L_4 ]

이러한 Layer 선택은 배열 방식과 무관한 함수의 본질적 특성이다.

반면 카르노맵 위에서 어떤 모양으로 보이는지는 배열 방식에 의해 결정된다.

따라서 다음을 구분할 필요가 있다.

Exactly-2 함수는 Weight=2 상태 여섯 개를 선택한다.

0132 배열 

Gray Code 배열에서는 Weight=2 상태들이 띠 형태로 배치된다.

그 결과 인간은 이를 하나의 연결된 구조로 인식하며, 시각적으로 고리(Ring)에 가까운 형태를 느끼게 된다.

0123 배열 

일반 이진 배열에서는 동일한 여섯 상태가 다른 위치에 배치된다.

이 경우 0132 배열에서 나타났던 고리 형태는 약해지며, 패턴은 기울어진 띠 또는 분산된 구조처럼 보인다.

즉 Weight=2 상태 자체는 동일하지만, 시각적 형태는 달라진다.

이 결과는 다음 사실을 보여준다.

고리 패턴은 Weight=2 Layer만으로 결정되는 것이 아니다.

4. XOR 함수 비교 

XOR 함수는 홀수 Weight 층을 선택한다.

[ L_1 \cup L_3 ]

0132 배열 

Gray Code 배열에서는 인접 셀 간 Hamming Distance가 항상 1이다.

따라서 인접 셀로 이동할 때 Weight의 홀짝성이 반드시 반전된다.

XOR는 홀수 Weight만 선택하므로 인접한 두 셀은 항상 서로 다른 값을 가진다.

그 결과 체커보드 패턴이 형성된다.

0123 배열 

일반 이진 배열에서는 인접 셀이 반드시 Hamming Distance 1을 만족하지 않는다.

따라서 Gray Code에서 나타나던 체커보드 구조가 사라진다.

대신 XOR 함수는 두 개의 대각선 방향 대칭성을 드러내게 된다.

본 연구에서는 이를 이중대각 패턴(Double-Diagonal Pattern)이라 정의하였다.

6. 연구 가설 

본 연구는 다음 가설을 제안한다.

가설 1 

패턴의 근본 원인은 함수가 선택하는 Layer 구조에 존재한다.

가설 2 

패턴의 구체적인 형태는 배열 방식에 의해 결정된다.

가설 3 

0132 배열은 Layer 구조를 시각적으로 드러내는 특수한 표현 방식일 수 있다.

즉 Gray Code 배열은 단순히 논리 최소화를 위한 배열이 아니라, Boolean 공간의 구조를 인간이 인식하기 쉬운 형태로 투영하는 역할을 수행할 가능성이 있다.

7. 소결론 

본 연구 결과는 카르노맵의 시각 패턴이 함수의 성질과 표현 방식의 성질이 결합되어 형성됨을 보여준다.

Layer 구조는 패턴의 원인을 제공한다.

배열 방식은 패턴의 형태를 결정한다.

따라서 카르노맵에 나타나는 고리 패턴, 체커보드 패턴, 이중대각 패턴은 단순한 배열 결과가 아니라 Boolean 공간 구조와 표현 방식의 상호작용 결과로 해석할 수 있다.
